ELEX Request a Quote plugin allows your customers to add products to a quote list, fill out a form, and request a custom price.

== Description ==
Boost Sales with Customized Quote Requests for WooCommerce Products! Easily implement a 'Request a Quote' feature, tailor-made for your customers. Effortlessly design and personalize quote request forms directly on your website's frontend. Our plugin ensures seamless communication with automated email notifications for quote submissions, approvals, and rejections, enhancing your customer engagement and streamlining your sales process.

= Features =
* Easily add Request a Quote option for WooCommerce Products.
* Supports both Simple and Variable Products.
* Hide the Add to Cart button.
* Customize WooCommerce Request a Quote from the way you want.
* Trigger Email notifications to the Customers and Admin.
* Intutive UI and Customisation Options

== Installation ==

1. Upload the plugin folder to the /wp-content/plugins/ directory.
2. Activate the plugin through the Plugins menu in WordPress.
3. That's it ! you can now configure the plugin.
